* Talk Abstract

Demonstrate how Babashka combined with Emacs org-mode enables literate
programming for DevOps tasks. Show how to create executable documentation
that combines narrative explanation, code, and Mermaid diagrams in a single
source file.

* Key Points

- Literate programming principles for infrastructure code
- Using org-babel with Babashka for reproducible scripts
- Integrating Mermaid diagrams for system visualization
- Real-world examples: deployment pipelines, monitoring scripts

* Target Audience

DevOps engineers, SREs, and developers interested in documentation-as-code
and reproducible infrastructure management.

* Outline (20-30 minutes)

1. Introduction to literate programming (3 min)
2. Org-mode + Babel setup (5 min)
3. Live demo: Building a deployment script (10 min)
4. Mermaid integration for system modeling (5 min)
5. Best practices and tips (5 min)
6. Q&A (2 min)
